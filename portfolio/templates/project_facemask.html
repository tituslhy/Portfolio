This project came at the height of the COVID-19 pandemic, and was aimed at (i) developing a light-weight image classification model that can detect if people were wearing masks, (ii) deployed on the edge [such as on thermal scanners], (iii) with a CI/CD MLOps pipeline that can automatically force model retraining if misclassification accuracy is too high. This project was coded on Google Cloud Platform. We also explore quantization of models to reduce deployment footprint.

Technologies used

  • Tensorflow
  • Tensorflow-Lite
  • Docker
  • VertexAI (Google Cloud Platform)
  • Data Studio (Google Cloud Platform)
  • Cloud Scheduler (Google Cloud Platform)
  • Big Query (Google Cloud Platform)
