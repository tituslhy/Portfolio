This project is my first foray into the field of automatic speech recognition and generative AI from large language models. The idea of this tool was to write a funny/meaningful birthday card from recorded video messages (previously and/or current).
This tool comprises of:

  • A video stitcher: Stitches all recorded videos into a single video. This can be used for data pre-processing or used to stitch a compilation as a gift.
  • An automatic speech recognition module: Using OpenAI's whisper model, we transcribe the meaningful messages into text for the LLM to do its work.
  • A generative AI module: Using langchain, we call the LLM (ChatGPT/HuggingFace models/other langchain supported LLMs) to do something funny - like write out an entire dating profile of your friend in html using the video transcript from Whisper. This can then be given as a gift to your friend as well

    • Technologies used

    • ChatGPT
    • HuggingFace
    • Whisper
    • Video processing
